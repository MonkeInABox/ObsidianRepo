## Hyper Text Markup Language:
- General layout of documents
- Clear specifications on error handling
- Not breaking the web (backward compatibility)
- Aiming at easier authoring of web apps

## Basic Syntax:
```
<!DOCTYPE html>
<html>
<head>
	<title>Page Title</title>
</head>

<body>
	<h1>Heading</h1>
	<p>Paragraph</p>
</body>
</html>
```
